Java-I/O

#BIO
InputStream、OutputStream(基于字节流，读取二进制文件)文件流、数据流、对象流
Reader、Writer（基于字符流，读取文本文件）

传统I/O是阻塞式的，当我们调用reader或者writer的时候，再有数据可以读取前，线程都是被阻塞的
NIO可以在等待数据时，让线程不阻塞，继续做其他的事情
