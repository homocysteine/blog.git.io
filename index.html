summary

By the way:if you want to check the effect of your edition,you must set up your hexo first,use the order like

1.title

If you want to write a title,you should type several ‘#’ before your content, the less ‘#’ there are, the highest priorty the title possesses.
Be careful that there must be a blank between the ‘#’ character and your title content.

2.enter

when you edit your blog, it is similar to edit a text in word. The enter and blank are also work.

3.order

If you want to show a code use special call to enclose your content.

4.website address

If you want to put on a website address, use the formal like
“[describe]”+”(the address)”.

5.picture

You have to modify the property of your deployment at first.
Change the “post_asset_folder” to true.
then use

Then create a new folder in source, the folder’s name must be consistent with your .md file.
put a picture into the folder, then use the formal like
“![]”+”(foldername/picturename.pictureformat)” to create it in the web page.
