Something you should know in run goldbug

Install the react-viro

execute the order in the root directory of your current project with

execute android-setup.sh

Find the script named android-setup.sh in node_modules->react-viro
Then , most importantly,you must move the script to the root directory of your project!!!!
In this way, you can either open the file with gitbash or cygwin64.

try to compile your project in Android Studio

one thing you should know in any time:
the version of buildtool,the version of gradle and the version of plugin must match each other.
You can modify your plugin version in
android->build.gradle
You can modify your gradle version in android->gradle->wrapper->gradle->wrapper.properties
You can modify your buildtool version in
android->app->build.gradle
